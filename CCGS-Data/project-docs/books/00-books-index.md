## 使用约定

1. **新书录入**：在对应子目录下放入章节 MD 文件，并创建 `00-book-digest.md` 摘要
2. **摘要编写**：包含书籍元信息、核心论点、章节导读表、方法论提取
3. **关联度标记**：⭐⭐⭐ 高（直接适用）/ ⭐⭐ 中（间接参考）/ ⭐ 低（远期储备）
4. **引用规范**：Phase 1 提案中引用方法论时，标注「出处：《书名》技巧 N / 第 N 章」
5. **优先级原则**：优先引用高关联度条目，但不排斥低关联度——在特定场景下任何方法论都可能成为关键洞察
