# 本书的论述逻辑

2 本书的论述逻辑1）爆款研究唐太宗李世民曾在《帝范》中列出帝王的12条大纲，其中1纲为：​“取法于上，仅得为中；取法于中，故为其下。自非上德，不可效焉。

​”从这个思路出发，我持续研究爆款游戏已经十多年了。我从大学学生时代起就持续的关注游戏市场上的爆款游戏，并对背后的一些设计改进过程与理念进行研究，查阅了大量的版本资料与开发者访谈。

因此我虽然并没有做出过爆款，却对爆款的理解极深。研究爆款并不意味着我们要去做爆款，而是可以从中取法，取法于上得乎其中。

正如同当我们去学习“第一性原理”​，就算我们不能造出可回收火箭，我们也可以让自己的实践能力更强。当我们向爆款游戏学习时，我们也可以从中获得大量通用型的方法论与技巧，用于自身游戏质量的提高。

反之，按照“自非上德，不可效焉”的原则，非爆款游戏学习价值有限，我们应该优先取法爆款游戏，因为它们的成功更能指明有效的游戏设计策略和市场趋势。固然爆款的成功有着极大的不可复制性，但爆款在制作与调控过程中所用的个别方法，也有着极大的可复制性。

通过深度的学习这些，我们可以提高自己在游戏设计领域的专业水平。

#### 2）3大核心因而在这本书中，我从爆款出发，对爆款进行了大量的分析研究，得到了不少策略与见解，我将之形成了爆款的方法论与实践技巧。

所谓的方法论是指用系统的、理论化的方法处理问题或进行研究。它涉及的是整个研究或工作流程的设计和理论框架，包含一系列的原则、理论和技巧。

我将爆款在游戏设计方面的方法论总结为3大核心，他们分别是游戏在玩法方面的处理，游戏在市场方面的处理，和游戏在玩家方面的处理。具体到论述的内容上，围绕如何将游戏做好玩，让游戏可以以最高的性价比与短视频、短资讯、流媒体、小说动漫等娱乐产品竞争，我撰写了第一卷，玩法设计卷。

围绕如何使用某种模式赚更多钱，依据市场情况，将公司战略、运转模式、运营考量、项目方向、研发团队、资源周期统筹在一起，我撰写第二卷，制作模式卷。围绕买量价格的日益上涨及其对高留存、高投资回报率的背景。

聚焦如何让产品在玩家方面适应市场变化，更好地满足用户需求，并扩大用户群。我撰写了第三卷，玩家调控卷。

这三卷三位一体互为补充，对于这3大核心，我们应该同等重视。为了论述的逻辑性，每卷有少量内容互相交织。

具体而言，每一卷的内容概述如下。

#### 3）第一卷概述本书的第一卷“玩法设计卷”​，解决的是如何设计一个好玩的玩法。

阅读完该卷内容后，读者可以将游戏设计的底层原理与自己设计的玩法结合，让游戏更加好玩。随着游戏行业进入存量市场，是什么让玩家决定玩A游戏而不玩B游戏呢？

我总结出如下公式：娱乐性价比=（游戏体验/游戏成本）×玩法新颖系数玩家会选择对他而言娱乐性价比最高的一款产品。其中的一个要点就是玩法新颖系数。

假如一个玩法，玩家已经在其他游戏中体验过一次，那么他在玩同质化抄袭游戏时，其感受到的娱乐性价比就会下降，其结果往往是抄袭产品难以重复原品的成功。这个问题，最好的解决方案就是对玩法进行创新，让玩家觉得游戏更“好玩和耐玩”​。

但创新并不是简单地做的不一样，要想做出一个既有创意又很好玩的玩法很难，需要探究游戏的制作原理，从底层分析和重新架构一个好玩的游戏。因此，我从玩家为什么会玩游戏、是什么让玩家觉得好玩、玩家会根据什么选择玩的游戏出发，分析游戏性的产生。

之后详细地阐述了如何利用各种系统机制构筑一个复杂的玩法模型，最终从零架构一个好玩的创新游戏。本书从最难的创新去论述游戏设计的底层，但假如你只是想调优体验，该模块内容也同样适用，毕竟连最难的问题都能解决，用其中的技巧去优化小的点也会轻而易举。

书我用了大量科学理论/游戏实例/数理分析的方法，帮助读者理解与印证玩法设计理论，并构筑了一些简要的模型与工具，让读者可以快速地套用到各个类型的游戏之中。通过第一卷所论述的各种通用底层原理，读者可以更好地进行玩法的调整。

本卷内容适合国内外独立的游戏小团队或想要调整游戏体验的开发者。

#### 4）第二卷概述本书的第二卷“制作模式卷”​，解决的是如何规划项目的整体模式。

阅读完该卷内容后，读者可以将具体的玩法设计与不同的制作模式相结合，让游戏开发具备更高的性价比。我们可以用第一卷的方法去做玩法创新，但创新也不是越多越好，创新的限度是什么，我们在什么情况下创新，我们如何确定根据自己的情况选择哪种游戏制作模式，第二卷是第一卷的整体扩展与衍生。

我们应该找到一个目标市场，然后选择最具性价比的方法，制作一款适宜的新游戏。一个游戏的成功是受多方面要素影响的，不是玩法好玩就行。

更重要的是玩家喜欢，并赚到足够的钱。游戏到底是一种电子产品，作为产品而言，应该能获得足够的利润，回收制作的成本，并让公司继续开发下一款产品。

要想有足够的利润，需要尽可能地提高游戏的营销收入，降低游戏的制作成本。而这些很大程度是你的制作模式，以及所选定的用户群决定的。

你的游戏内容是借鉴还是创新的，需要多大的开发与调试工作，目标用户群的规模与付费能力如何，需要多大的推广成本买入用户。还有其他诸如商业模式等因素，这些综合决定了你的游戏最终能否赚钱。

但赚钱只是对于研发方而言，玩家最终看的依然是娱乐性价比，我们需要在这两者间取一个相对平衡。玩法创新是提高娱乐性价比最好的方法，但也有着代价巨大的缺陷，且在研发方面有着极大的不可控风险。

综合来看，在其他方面创新/体验调优，也可以提供不错的效果，让玩家选择你的游戏并付费。而这些非玩法方面的创新/体验调优点，也需要有重点、有限制地使用。

我们一定要搞清楚何处应该创新，何处又应该借鉴，这需要考虑不同的游戏制作模式，看哪一种更适合你目前所在的研发方。因此第二卷一共论述了4种游戏制作模式，详尽地分析了它们的优缺点，并提出了一些选择与制作要点，帮助研发方分析市场情况和公司优势，找到某个市场做一款对研发方而言性价比更高的游戏。

通过阅读此部分内容，你可以更清晰地认识到自己现在应该做一款什么样的游戏，应该往哪方面发力。在书中我分析了多个爆款游戏，又以手机行业的兴衰成败作为印证，说明了在企业层次，应该如何选择开发制作模式，如何有限度地创新，以更好地把握未来。

本卷内容适合国内体量中大的游戏公司或其中的策划。

#### 5）第三卷概述本书的第三答卷“玩家调控卷”​，解决的是如何调控出一款大众化的游戏。

阅读完该卷内容后，读者可以将各种制作模式与不同类型的玩家追求结合，让游戏的留存与付费数据更好。游戏行业的推广大部分时候依然需要买量，各种类型的用户都有，非常杂。

这就导致了众口难调，无论是刚上线的游戏，还是运营很久的游戏，都需要解决如何在游戏中让更多的玩家满意，以获得更大的DAU，更好的留存数据。这需要把你的游戏与各类型玩家巧妙结合。

但各类型玩家的需求往往是相互冲突的，如果从用户需求出发，符合所有玩家其实就是不符合所有玩家。我认为这个问题的解决方法应该是：通过详尽的用户行为研究，从用户的追求出发，详尽地分析用户属性和追求路径，考虑到多个环节的转化与游戏生态变化，最终达到兼容。

通过这种方法不只可以用于调优复杂的重度游戏，对于简单的单机休闲手游/微信小游戏也同样适用。在书中我详尽地阐述了这套方法论，并以科学研究作为基础知识，从多个不同类型的爆款游戏作为例证（轻度手游天天过马路、中度手游梦幻模拟战、重度端游魔兽世界）​，帮助玩家更好地认知适应这套理论，把握住表现为用户行为的用户需求。

你对用户行为把握得越精确，你就越可以用一些极简的小模块，用较小的开发量，调优出一款玩家众多的游戏。本卷论述的内容，在游戏的全时期，尤其是即将上线和上线以后的版本更新阶段，都有着极高的价值。

它将协助你以上帝视角透视多类用户的“后台”​，从而找出局部最优解，调优自己的游戏。本卷适合国内外各种类型的研发项目和发行/运营岗位人员。

在全书的最后两章，我还额外论述了游戏与现实的对比。抽象总结了游戏化的相关内容，并对比了游戏面临的其他娱乐竞品，如短视频的竞争优劣势分析，得出了破局的方法，相信对于各类读者都将大有裨益。
