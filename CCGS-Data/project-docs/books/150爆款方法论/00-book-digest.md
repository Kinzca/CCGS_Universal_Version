# 《爆款方法论》书籍摘要 (Book Digest)

> **书名**：爆款方法论：游戏设计的3大核心与150个技巧  

> **作者**：曾宣凯  

> **出版**：2024年3月 / 51.2万字  

> **领域**：游戏设计方法论（玩法 / 制作模式 / 玩家调控）  

> **适用角色**：🎮 Game Designer / 🗺️ Level Designer / 🔬 UX Researcher  

> **总体评价**：以战略学思维剖析游戏设计，提供 150 个可操作的设计技巧，覆盖从底层原理到用户行为的全链路。

三卷递进结构（玩法→制作→调控）对独立游戏和解谜品类高度适用。

---

## 核心论点

1. **游戏是高维变色龙**：不同主体视角呈现不同运行姿态，需用正确方法对不同姿态研究

2. **灵魂→机制→表现**：设计时从灵魂出发保证内在一致性；分析时用机制→运行→体验逆向拆解

3. **6级设计优先**：美 > 自然 > 道 > 法 > 术 > 器，下级服从上级

4. **娱乐性价比公式**：娱乐性价比 = (游戏体验/游戏成本) × 玩法新颖系数

5. **深度优先原则**：先纵深打透核心体验，再横向扩展
