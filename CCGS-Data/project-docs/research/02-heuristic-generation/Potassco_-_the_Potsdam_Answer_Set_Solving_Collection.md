# Getting Started

Answer Set Programming (ASP) offers a simple and powerful modeling language to solve combinatorial problems. With our tools you can concentrate on an actual problem, rather than a smart way of implementing it. [Get started!](https://potassco.org/doc/start/)

To get a quick first impression, you may want to experiment with [running clingo](https://potassco.org/clingo/run/) in your browser.

# Systems

To find out more about a specific system and a download link, follow one of the links below.

[clingo](https://potassco.org/clingo/) is an ASP system to ground and solve logic programs.

[gringo](https://potassco.org/clingo/) is a grounder (powering the grounding in clingo).

[clasp](https://potassco.org/clasp/) is a solver (powering the search in clingo).

[clingcon](https://potassco.org/clingcon/) extends clingo with constraint solving capabilities.

[aspcud](https://potassco.org/aspcud/) is a solver for package dependencies.

[asprin](https://potassco.org/asprin/) is a general framework for qualitative and quantitative optimization in ASP.

[clingraph](https://potassco.org/clingraph/) is a tool for visualizing answer sets.

[clinguin](https://potassco.org/clinguin/) is a tool for creating User Interfaces in ASP.
