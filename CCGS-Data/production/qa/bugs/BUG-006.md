## Bug Report
- **ID**: BUG-006
- **Title**: TowerState/GateState 编译报错 CS1061 (缺少 TickMiddleButtonDrag)
- **Severity**: S1
- **Frequency**: Always
- **Build**: commit 9665324
- **Platform**: Unity Editor

### Steps to Reproduce
1. 执行 Story 001 的代码重构，在 `RewiredMouseDragProvider.cs` 中移除了 `TickMiddleButtonDrag`。
2. 触发 Unity 编译。
3. 查看 Unity Console。

### Expected Behavior
代码无缝编译通过，没有遗留未清理的引用。

### Actual Behavior
出现两处编译错误：
`Assets\GameMain\Scripts\HotFix\GameLogic\GameFlow\TowerState.cs(220,18): error CS1061: 'RewiredMouseDragProvider' does not contain a definition for 'TickMiddleButtonDrag' ...`
`Assets\GameMain\Scripts\HotFix\GameLogic\GameFlow\GateState.cs(72,18): error CS1061: 'RewiredMouseDragProvider' does not contain a definition for 'TickMiddleButtonDrag' ...`

### Additional Context
在进行 Story 001 相关的输入层清理时，移除了 `TickMiddleButtonDrag` 但没有通知在状态机（TowerState 和 GateState）中调用它的代码。这两个状态机的 `OnUpdate` 中应该有手动调用的痕迹，需要将其删除或修改，以适配全新的自动视角旋转流。
