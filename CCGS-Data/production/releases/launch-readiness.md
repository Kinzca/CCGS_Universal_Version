## 维度评估标准

### 核心玩法完整度（30%）

| 评分区间 | 标准 |
|:---|:---|
| 90-100 | 核心循环完整，所有设计支柱已实现，心流体验达标 |
| 70-89 | 核心循环可玩，部分设计支柱待完善 |
| 50-69 | 核心循环基本可用，存在明显体验缺口 |
| <50 | 核心循环不完整，阻断上线 🔴 |

### 技术稳定性（25%）

| 评分区间 | 标准 |
|:---|:---|
| 90-100 | 崩溃率 <0.1%，无严重 Bug，Bug Tracker 清空 |
| 70-89 | 崩溃率 <0.5%，无严重 Bug，少量中等 Bug |
| 50-69 | 崩溃率 <1%，存在中等 Bug 但有绕行方案 |
| <50 | 崩溃率 ≥1% 或存在严重 Bug 🔴 |

### 性能达标率（15%）

| 评分区间 | 标准 |
|:---|:---|
| 90-100 | 目标设备全部 ≥60fps，加载 <3s |
| 70-89 | 目标设备 ≥30fps，加载 <5s |
| 50-69 | 部分场景掉帧，加载 <10s |
| <50 | 频繁掉帧或加载 >10s 🔴 |

### 内容完成度（15%）

| 评分区间 | 标准 |
|:---|:---|
| 90-100 | 计划内容全部完成，打磨到位 |
| 70-89 | 核心内容完成，边缘内容待完善 |
| 50-69 | 核心内容基本可用，有占位资源 |
| <50 | 大量内容缺失 🔴 |

### 本地化就绪度（10%）

| 评分区间 | 标准 |
|:---|:---|
| 90-100 | L3 发布就绪，至少两种语言完成 |
| 70-89 | L2 翻译就绪，主语言完成 |
| 50-69 | L1 基础就绪，文本已外部化 |
| <50 | L0 未就绪，存在硬编码文本 |

### 文档完备度（5%）

| 评分区间 | 标准 |
|:---|:---|
| 90-100 | GDD 完全一致，发布说明齐全 |
| 70-89 | GDD 基本一致，有发布说明 |
| 50-69 | GDD 部分过时，有基础发布说明 |
| <50 | GDD 严重过时，无发布说明 |
